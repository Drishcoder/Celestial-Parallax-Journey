# Celestial Parallax Journey

A stunning interactive web experience featuring smooth parallax scrolling, custom cursor tracking, immersive audio synthesis, and cinematic visual effects.

## Features

- 🎨 Smooth parallax scrolling with lerped animations
- 🔊 Ambient synthesizer soundtrack with section-based modulation
- ✨ Interactive custom cursor and scroll progress tracking
- 🎭 Advanced visual effects: curtain reveals, asymmetric layouts, and gradient morphing
- 📱 Fully responsive design with touch optimization
- ⚡ High-performance WebGL and CSS animations

## Getting Started

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open http://localhost:3000 in your browser

## Building

```bash
npm run build
```

The optimized production build will be output to the `dist/` directory.
